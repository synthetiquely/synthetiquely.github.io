Мои работы

CarFinder

github.com

Приложение, которое позволяет аггрегировать все актуальные объявления по продаже автомобилей с крупнейших белорусских платформ. Приложение предоставляет фильтры для поиска и показа актуальных объявлений на основе пожеланий пользователя. Также можно сохранять параметры фильтров, чтобы получать уведомления по электронной почте о новых предложениях. Приложение также предоставляет возможность проверить ликвидность автомобиля по модели и типу кузова.

  • TypeScript
  • React
  • Redux
  • Bulma
  • KoaJS
  • MongoDB

Task Tracker

github.com

Приложение для организации работы над ИТ-проектами. Возможность зарегистрироваться как менеджер проекта либо разработчик. Менеджеры проекта могут создавать новые проекты, добавлять существующих людей к ним, и управлять работой над ним. Разработчики могут создавать и выполнять задания, писать комментарии к заданиям.

  • VueJS
  • Vuex
  • Semantic UI
  • Nodejs
  • MongoDB

Survey App

github.com

Приложение для организации и проведения опросов. Имеет встроенный конструктор для создания опросов различной сложности. Созданный опрос также можно сохранить как шаблон для использования в будущем. Позволяет искать уже созданные опросы. Предоставляет платформу для проведения и анализа результатов проведения опросов.

  • React

  • Express
  • MongoDB
